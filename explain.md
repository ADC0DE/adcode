# 애드코드(ADCODE) 웹사이트 구조 설명서 (크몽 전문가해빙)

## 메인 페이지 (브라우저 주소창에 직접 입력되는 파일들)
웹사이트의 각 메뉴를 클릭했을 때 실제로 보여지는 개별 페이지들입니다.

*   **`index.html`** : 웹사이트의 첫 화면(홈)입니다. 회사 소개와 전체적인 서비스 요약을 담고 있습니다.
*   **`viral.html`** : '바이럴 마케팅' 서비스에 대한 상세 설명 페이지입니다.
*   **`offline.html`** : '가게(매장) 마케팅' 서비스에 대한 상세 설명 페이지입니다.
*   **`performance.html`** : '퍼포먼스 마케팅' 서비스에 대한 상세 설명 페이지입니다.
*   **`design.html`** : '디자인 제작' 서비스에 대한 상세 설명 페이지입니다.

## 공통 부품 폴더 (`components`)
모든 페이지에서 매번 똑같이 사용하는 부분들을 따로 떼어서 관리하는 곳입니다. 이곳의 파일을 하나만 고치면 모든 페이지의 메뉴나 하단 정보가 한 번에 바뀝니다.

*   **`header.html`** : 모든 페이지 맨 위에 나오는 **로고와 상단 메뉴바**입니다.
*   **`footer-contact.html`** : 모든 페이지 맨 아래에 나오는 **문의하기 폼, 주소/사업자 정보, 이용약관 팝업창** 내용들이 들어있습니다.

## 디자인 및 기능 파일들 (`assets`)
웹사이트를 예쁘게 꾸미고, 움직이게 만드는 '재료'들이 모여있는 창고입니다.

*   **`assets/css/style.css`** : 웹사이트의 색상, 글자 크기, 간격 등 **모든 디자인 규칙**이 들어있는 파일입니다.
*   **`assets/js/main.js`** : 숫자가 올라가는 효과나 화면이 부드럽게 나타나는 애니메이션 등 **동적인 기능**들을 담당합니다.
*   **`assets/js/layout.js`** : 위에 설명한 상단 메뉴(`header.html`)와 하단 영역(`footer-contact.html`)을 각 페이지에 **자동으로 조립**해주는 똑똑한 안내원 역할을 합니다.

## 이미지 및 기타
*   **`adcode_logo.png`** : 웹사이트 상단에 보이는 공식 로고 이미지입니다.
*   **`adcode_fabicon.jpg`** : 인터넷 탭 창에 작게 표시되는 아이콘(파비콘)입니다.
*   **그 외 `.jpg` 파일들** : 웹사이트 배경이나 각 섹션에서 사용되는 고화질 사진들입니다.

**참고**: 루트(가장 바깥쪽)에 있는 `style.css`는 리팩토링 이전의 백업 파일이며, 실제 현재 디자인은 `assets/css/style.css` 파일을 통해 작동하고 있습니다.
